# 负样本欠采样模块实现文档

## 概述

本模块实现了针对Tick级涨停预测任务的负样本欠采样策略，有效解决了极端数据不平衡问题（原始比例约1:77.9），将训练集正负样本比例调整为1:5。

## 实现文件

### 1. 核心模块
**文件路径**: `src/data_processing/sampling.py`

包含以下函数：
- `stratified_negative_sample()` - 第一层采样：基于dist_to_limit分层筛选
- `balance_sampling()` - 第二层采样：随机采样控制比例
- `two_layer_negative_sampling()` - 完整两层采样流程
- `undersample_train_set()` - 训练集封装函数
- `get_sampling_statistics()` - 获取采样统计信息
- `validate_sampling_result()` - 验证采样结果
- `automatic_undersample_if_needed()` - 自动检测并采样

### 2. 模块导出
**文件路径**: `src/data_processing/__init__.py`

已添加采样模块的导入和导出。

### 3. 集成点
- **build_2025_dataset.py** - 在`build_full_year_dataset()`函数中集成
- **scripts/dataset_builder.py** - 在`DatasetBuilder._split_dataset()`方法中集成
- **train_model.py** - 在`train_from_split_datasets()`函数中添加自动检测

## 采样策略

### 第一层采样：分层筛选
基于`dist_to_limit`（距离涨停价相对距离）进行分层：

- **接近涨停**（dist_to_limit ≤ 1%）：保留 **100%**
- **中等距离**（1% < dist_to_limit ≤ 5%）：保留 **30%**
- **远离涨停**（dist_to_limit > 5%）：保留 **5%**

### 第二层采样：比例控制
随机采样调整到目标比例 **1:5**（负样本:正样本）

## 关键特性

1. **仅训练集采样**：严格遵循技术文档要求，验证集和测试集保持原始分布
2. **正样本100%保留**：确保所有涨停信号都被模型学习
3. **固定随机种子**：使用随机种子42确保可重复性
4. **特征完整性**：采样后保持所有特征列不变
5. **自动检测**：训练脚本中包含自动检测功能，当比例>1:8时自动采样

## 使用方法

### 1. 在数据集构建中使用

```python
from src.data_processing.sampling import undersample_train_set

# 在数据集划分后、保存前执行
train_df = undersample_train_set(
    train_df,
    dist_col='dist_to_limit',
    label_col='label',
    thresholds=(0.01, 0.05),
    keep_ratios=(1.0, 0.3, 0.05),
    target_ratio=5.0,
    random_seed=42,
    verbose=True
)
```

### 2. 在训练脚本中使用（自动模式）

```python
from src.data_processing.sampling import automatic_undersample_if_needed

# 自动检测并采样
train_df = automatic_undersample_if_needed(
    train_df,
    dist_col='dist_to_limit',
    label_col='label',
    auto_threshold=8.0,  # 当比例>1:8时自动采样
    thresholds=(0.01, 0.05),
    keep_ratios=(1.0, 0.3, 0.05),
    target_ratio=5.0,
    random_seed=42,
    verbose=True
)
```

### 3. 使用完整两层采样流程

```python
from src.data_processing.sampling import two_layer_negative_sampling

sampled_df = two_layer_negative_sampling(
    df,
    dist_col='dist_to_limit',
    label_col='label',
    thresholds=(0.01, 0.05),
    keep_ratios=(1.0, 0.3, 0.05),
    target_ratio=5.0,
    random_seed=42,
    verbose=True
)
```

## 注意事项

1. **不要对验证集/测试集采样**：这会导致评估结果不准确
2. **保持随机种子一致性**：使用相同随机种子确保可重复性
3. **检查必需列**：确保数据集包含`dist_to_limit`和`label`列
4. **监控采样效果**：使用`validate_sampling_result()`验证结果

## 技术参数

| keep_ratios | (1.0, 0.3, 0.05) | 各层保留率 |
| target_ratio | 5.0 | 目标正负比例 |
| random_seed | 42 | 随机种子 |
| auto_threshold | 8.0 | 自动采样阈值 |
