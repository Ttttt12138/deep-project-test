评估口径差异

H1 tuned 测试集正样本率约 16.67%；Rolling CV 验证集是百万级真实候选池，正样本率约 0.02%。PR-AUC 不能跨口径直接等价比较。

实际使用侧重点

少量高置信筛选看 Daily Precision@10；真实候选池上线前重点看 Rolling PR-AUC、P@100、P@500 和按月稳定性。
