## 技术特点

### 1. 首次触板检测
```python
# 向量化提取所有触板的行索引
touch_indices = df[all_touch_events == 1].index

# 按代码分组，取每个代码的第一个触板索引
first_touch_indices = df.loc[touch_indices].groupby('code').head(1).index
df.loc[first_touch_indices, 'label'] = 1
```

### 2. 智能NaN填充
```python
# 价格类特征：使用向前/向后平推（避免未来函数泄露）
if is_price_like:
    df[col] = df[col].ffill().bfill()
    if df[col].isna().any():
        df[col].fillna(0, inplace=True)
else:
    # 成交量/变化率特征：用0填充
    df[col].fillna(0, inplace=True)
```

- **建模目标**：以盘口前10个Tick的动态趋势，预测第11个Tick是否会发生"首次触板"事件

- **预期效果**：单日正样本约10-80个，纯度100%
